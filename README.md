# BTC Liquidations Monitor

Real-time Bitcoin liquidation tracking from Binance Futures.

## Features

- Live WebSocket connection to Binance Futures
- Track Long and Short liquidations in real-time
- Configurable threshold filters
- Visual charts showing liquidation volume
- Auto-reconnect on connection loss

## Tech Stack

- Next.js 14 + TypeScript
- Tailwind CSS
- Recharts
- Binance WebSocket API
